h4. The project

The aim of this project is to implement a lightweight IOC container who respects the  "JSR 330":http://code.google.com/p/atinject/ in Java. To use it you have to download the source and install the project in your maven repository with *mvn install*.

h4. Maven dependencies

* the core module

<pre>
<groupId>com.my.container</groupId>
<artifactId>container-core</artifactId>
<version>0.1-SNAPSHOT</version>
</pre>

* the aop module

<pre>
<groupId>com.my.container</groupId>
<artifactId>container-aop</artifactId>
<version>0.1-SNAPSHOT</version>
</pre>
