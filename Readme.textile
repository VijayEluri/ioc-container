h2. Readme File 

h4. The project

The aim of this project is to implement a lightweight IOC container who implements the  "JSR 330":http://code.google.com/p/atinject/ in Java. To use it you have to download source and install it in your maven repository with <code>mvn install</code> command

h4. Maven dependencies

* core module

<pre>
<code><groupId>com.my.container</groupId>
<artifactId>container-core</artifactId>
<version>0.1-SNAPSHOT</version></code>
</pre>

* aop module

<pre>
<code><groupId>com.my.container</groupId>
<artifactId>container-aop</artifactId>
<version>0.1-SNAPSHOT</version></code>
</pre>

h4. Requirements

* JDK5 or later
